Joining GitHub

Digital Justice Emails Addresses

Users with a @digital.justice.gov.uk email address can join the MoJ GitHub Organization using the Single Sign On https://github.com/orgs/ministryofjustice/sso link.

Please also complete this form for our records.

Pre-Approved Emails Addresses

Users with one of the following approved email addresses can join the MoJ GitHub Organization:

  • @digital.justice.gov.uk
  • @justice.gov.uk
  • @publicguardian.gov.uk
  • @cica.gov.uk
  • @ima-citizensrights.org.uk

Users with one of the following approved email addresses can join the analytical-services GitHub Organization:

  • @digital.justice.gov.uk
  • @justice.gov.uk
  • @publicguardian.gov.uk
  • @judicialappointments.gov.uk
  • @judiciary.uk
  • @ppo.gov.uk
  • @sentencingcouncil.gov.uk
  • @yjb.gov.uk

Complete the form to gain access.

Other Users

Complete the form to request MoJ GitHub Organization and/or analytical-services GitHub Organization access.

External Users

External users can get access to an Organization repository as an Outside Collaborator.

For access to a MoJ GitHub Organization repository see https://github.com/ministryofjustice/github-collaborators.

For access to a analytical-services GitHub Organization repository see https://github.com/moj-analytical-services/github-outside-collaborators.
