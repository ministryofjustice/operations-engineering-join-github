Joining GitHub

The Ministry of Justice maintains two GitHub Organizations,

  • Ministry of Justice
  • MoJ Analytical Services

Please see below for instructions on how to join.


Digital Justice Email Addresses

Users with a @digital.justice.gov.uk email address can join the Ministry of Justice GitHub Organization using Single Sign On.

Approved Email Addresses

Users with one of the following approved email addresses can join the Ministry of Justice GitHub Organization:

  • @digital.justice.gov.uk
  • @justice.gov.uk
  • @publicguardian.gov.uk
  • @cica.gov.uk
  • @ima-citizensrights.org.uk

Users with one of the following approved email addresses can join the MoJ Analytical Services GitHub Organization:

  • @digital.justice.gov.uk
  • @justice.gov.uk
  • @publicguardian.gov.uk
  • @judicialappointments.gov.uk
  • @judiciary.uk
  • @ppo.gov.uk
  • @sentencingcouncil.gov.uk
  • @yjb.gov.uk

Complete this form to gain access.

Other Users

Create a request on the Operations-Engineering team Slack channel #ask-operations-engineering. Please provide which GitHub Organisation/s you would like to join and your email address.

External Users

External users can get access to an Organization repository as an Outside Collaborator.

For access to a Ministry of Justice GitHub Organization repository see https://github.com/ministryofjustice/github-collaborators.

For access to an MoJ Analytical Services GitHub Organization repository see https://github.com/moj-analytical-services/github-outside-collaborators.
